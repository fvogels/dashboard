# Dashboard

Used as new-tab page on your browser.

## Running locally

```bash
$ npx vite run --mode MODE
```

where MODE is the dashboard you want to serve, for exmaple `npx vite run --mode home`.

## Building

```bash
$ npx vite build --mode MODE
```

## Creating new dashboard

Define new `Dashboard` under `src/dashboards`.
In `App.tsx`, add case.
